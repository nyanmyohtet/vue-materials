## Vue CLI

https://cli.vuejs.org

### Installation

Run the following command in PowerShell to install VueCLI:

```
npm install -g @vue/cli
```

![VueCLI Install.png](_resources\vue-cli-install.png)

Create a Vue project:

```
vue create my-project
# OR
vue ui
```
